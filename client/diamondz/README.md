# Diamondz - Paint Protection Film Services

Diamondz is a modern, responsive web application built with Angular for showcasing and managing Paint Protection Film (PPF) services. The application provides information about various PPF options, company details, services offered, and a way for customers to get quotes and contact the business.

## Features

- **Home Page**: Hero section with company branding and key highlights.
- **About**: Information about the company and its vision.
- **Services**: Detailed description of PPF services including gloss, matte, and colored options.
- **Gallery**: Showcase of completed PPF installations.
- **Process**: Step-by-step guide on how PPF is applied.
- **Team**: Meet the team behind Diamondz.
- **Partners**: List of partners and collaborations.
- **FAQ**: Frequently asked questions about PPF.
- **Contact**: Contact form and information.
- **Quote Request**: Form for customers to request quotes.
- **Responsive Design**: Optimized for desktop and mobile devices using Tailwind CSS.

## Technologies Used

- **Angular**: Framework for building the application.
- **TypeScript**: Programming language for type-safe development.
- **Tailwind CSS**: Utility-first CSS framework for styling.
- **PostCSS**: Tool for transforming CSS with JavaScript.
- **Vitest**: Testing framework for unit tests.
- **Angular CLI**: Command-line interface for Angular development.

## Installation

1. Clone the repository:
   ```bash
   git clone <repository-url>
   cd diamondz
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   ng serve
   ```

4. Open your browser and navigate to `http://localhost:4200/`.

The application will automatically reload when you make changes to the source files.

## Building for Production

To build the project for production, run:

```bash
ng build --configuration production
```

The build artifacts will be stored in the `dist/` directory.

## Running Tests

- **Unit Tests**:
  ```bash
  ng test
  ```

- **End-to-End Tests**:
  ```bash
  ng e2e
  ```
